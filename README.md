# Role: sh

Configure bash server-wide.

**Compatibility tested with:**
 * Debian stable
 * Fedora stable

## Configuration
| Name | Default value | Description |
|------|---------------|-------------|
| `sh_environment` | `null` | Environment to show in the shell (`{'prod', 'test'}`) |
| `xdg_env` | (see defaults/main.yml) | XDG environment variables |
